Getting Started

Welcome to AquaTouch!

Thank you for installing AquaTouch!
AquaTouch is a TouchBar add-on UI that replaces or adds more functionality to the native MacOS TouchBar.

Here you will find useful information about AQT, how to operate it, how to customise it, and find out about hidden features.

Opening and Closing the AquaTouch UI

AquaTouch is designed to work with the macOS Stock Touch Bar, therefore it may close to reveal stock controls or open to reveal it's own controls.

AquaTouch will open automatically for fully supported apps, and should close automatically for unsupported apps.

It's possible to manually switch between the stock Touch Bar and AquaTouch, you can;

Use Touch Bar Buttons:
  • In the stock Touch Bar, tap the AQT icon in the Control Strip to open AQT.
  • In AquaTouch, hold the apps icon to go back to the stock Touch Bar.

    Or, you can use the keyboard:
  • Give the [fn] key a quick tap to toggle between Stock or AquaTouch.

    • The different types of AquaTouch Layouts

    AquaTouch can open up to either of the three types of layout for different apps.
    The set that will open up can be noted by its icon on the control strip.

    Full Support

    A full set replaces an app's existing controls, it will open automatically.

    Extension

    This layout has a focus on stock touchbar controls, and will close automatically when not in use. It's main aim is to add a little bit more to existing, great stock controls.

    Not Supported

    Apps that AQT doesn't support look like this. You can still control the volume, screen brightness and keyboard brightness using gestures here, and access the QuickStrip.

    Volume, Brightness and Keyboard Backlight controls in AquaTouch

    AquaTouch uses Gestures in the TouchBar to control Volume, Brightness and Keyboard Backlight.

    Volume
    Slide on the touchbar with two fingers to control the Volume

    Brightness
    Slide on the touchbar with three fingers to control the Screen Brightness

    Keyboard Backlight
    Slide on the touchbar with four fingers to control the Keyboard Backlight

    Note:
    Due to limitations in macOS, AquaTouch needs to be open while you slide.
    Unfortunately I'm not able to make the gestures work in the stock touchbar, so you'll need to open AQT before doing a gesture. I still find this easier than the stock sliders, even though it takes a bit to get used to.


    FAQ

    How do I adjust the volume and display brightness?

    With AquaTouch we use gestures! Read above to find out how ;)


    Is there a mute button?

    Since we use gestures, simply slide two fingers on the touchbar all the way to the left.


    I see an extra digital escape key

    If you have a physical escape key, head to AQT Settings to disable the digital escape key.


    Why is the escape key cut off on tap?

    This is by design; it brings the escape key to the very left edge of the Touchbar, emulating where it normally is.
    No longer do you need to remember to shift your finger to the right a bit, you can now tap it as you've always done it. Think of the touchbar as a now bezel-less display.

    Unfortunatley since there is no actual screen to the left of that area, the button will look cut off when you press it. I can't find a way to hide this, so we'll just need to deal with it for now.
    If you would like the normal Touchbar escape look back, you can select the look you prefer in the AQT settings from the BTT Window's Touchbar.


    AquaTouch is very buggy for me, and a lot of things don't seem to work or respond.

    Aquatouch needs to read and interact with the MacOS UI, and since it was made on an English system, it looks for english words and menus. If MacOS isn't in english, it may have trouble finding some menus to interact with, causing bugs.

    If you're able to, switch your macOS system language to english, otherwise AquaTouch would need to be translated to work with your language.

    If your system is in English, it may be a bug. Report it on my Discord server or on the AQT Forums.


    AquaTouch Quick Strip

    AquaTouch Quick Strip

    The Quick Strip is the four-button pallete seen at the very right. It will always be available there for you to access from any aquatouch screen.

    It gives you access to helpful widgets, controls and shortcuts that are great to adjust what music you're listening to, adjust some mac settings, check your calendar and more. Here you can read a summary of its four segments.

    Quick Media

    About
    This Segment contains media controls and Now Playing widgets.
    If a supported app is playing media, they will show up within this segment.
    Additionally for supported apps, media buttons will directly control the playbackfrom that specific app.
    e.g. You can pause iTunes, then play Spotify, all from this segment, without clicking anything.

    Hold Action
    You can hold this segment button to Play/Pause Media.

    Icon Color
    The icon for this segment will turn blue if media is playing.

    Quick Schedules

    About
    Tap to check your claendar and reminders. You can also view the weather and time here.

    Hold Action
    Hold this control strip button to toggle Do Not Disturb

    Icon Color
    The icon for this segment will turn red if you have any upcoming Calendar or Reminders events.
    You can change its behaviour in AQT settings, e.g. set it to only turn red for Reminders.

    Quick Controls

    About
    Set system controls such as NightShift, Dark Mode or to sleep your MacBook. There's a connectivity pallete to the right that allows you to connect to AirPods, Beats, toggle WiFi, toggle Do Not Disturb and more.

    Hold Action
    Hold this control strip button to Open a Bluetooth Tool

    Icon Color
    The icon for this segment will turn violet if Do Not Disturb is active.

    Quick Apps

    About
    Tap this segment to launch a quick task with one button. You can hold command here to quit multiple apps with a touch, too.

    Hold Action
    Hold this segment button to switch to the stock MacOS Touch Bar.

    Understanding Icons and Symbols

    Holdable Buttons



    Holdable Buttons are indicated by a folded corner. Hold these buttons to see more options related to that button.


    Back Buttons



    Back buttons in AquaTouch tell you the group which you have open. Tap it to exit the group.

    If you open a group within a group, the back buttons will stack. Tapping a back button will close the group displayed on the button.


    Time Scrubbers



    Time scrubbers control playback of media.

    Drag the slider to control playback, or tap the time labels on either side of the slider to skip forward or back 15 seconds.

    Workflow Tools

    Workflow Tools

    AquaTouch adds some features outside of the touchbar that may make your experience in MacOS Better. Check out some of them below!

    Window Snapping

    Easily snap windows to left or right halves of the screen, or even any corner.

    Right click on the green window button or use fn-W to bring it's menu up.
    Right click the yellow window button to restore the old window size.

    If you have multiple monitors connected, you'll see an expanded menu that allows you to snap to ajacent monitors.

    Control Windows using Keyboard Shortcuts

    Use Keyboard Shortcuts to access the red, yellow and green buttons of windows.

    [ fn 0 ] (zero) - Close Window (Red button)
    [ fn - ] - Minimise (Yellow button)
    [ fn + ] - Full Screen (Green button)

    [ fn w ] - Activate Window Snapping Menu


    Modifier Action Suggestions

    In some apps, The Touchbar can suggest actions you may want to do when a modifier key is held down.

    Supported Apps:

  • Finder
  • Photoshop


    • Touchbar Widget: Click Pulse

    Some apps have a 'Click Pulse' tool which allows you to click an array of buttons such as a list of checkboxes or 'show layer' toggles.

    You'll see a "Hold" button in the Click Pulse menu. Hold it and AquaTouch will click the mouse rhythmically for you. Then, hover over the buttons or checkboxes and let Click Pulse click them for you.

    This can be useful to select multiple layers in Photoshop, Illustrator and selecting media to sync in iTunes.

    Try it here:

    Touchbar Widget: PasteCells

    Some apps have special widgets in their Touchbar UI called "PasteCells". It's an original innovation of AquaTouch that allows you to save snippets of text into the Touchbar so you can easily insert them into your document.

  • Hold a PasteCell to save text into it.
  • Tap a PasteCell to paste it's contents into your document.

    Customisation

    Some AquaTouch features can be customised to your liking!

    To find AquaTouch Settings, open the BTT configuration menu, Slide the white touchbar and hit "AQT Settings".
    Here you can change how the escape key looks, QuickStrip behaviour, and much more.

    These settings are saved and will cross over to future AquaTouch Versions!

    Notification Badge Customisation

    Notification Badges

    If supported, you can get a notification bubble in the touchbar when you have a notification for it.

    Head to the AquaTouch Settings to enable these for more of your apps!

    The AquaTouch Settings are found in the touchbar after opening the BTT Configuration Window.

    Advanced Customisation

    AQT Advanced Customisation

    In Addition to the AQT Settings accessed in the TouchBar, some additional options are available in the BTT configuration UI.

    Some things you can change at the moment:

  • Haptic Feedback Styles
  • Control Strip Button Hold Actions

    • How To: AQT Advanced Customisation

    1. Open the Better Touch Tool Window.
    2. Head to the 'Named and Other Triggers' from the top left device dropdown
    3. Select 'All Apps' in the leftmost sidebar.
    4. You'll see a AQT USER PREFERENCES section at the top of the trigger list (second column). Select the item you wish to Customise
    5. Change/tweak the 'Assigned Actions' to what you wish. For haptic options, simply select the action (third column) and select your preferred style in the fourth column.

    Update-Proof Customisation

    If you're a more advanced user, you may want to add your own widgets to AquaTouch. Unfortunatley you will loose these modifications if you add them the easy way. However, there is a way to make them move across AquaTouch updates! Here's how:

    About this Method

    BTT allows you to have more than one preset active at one time.
    This function can be used to seperate your modifications from AquaTouch, thus, your modifications could stay put. If modifications are added into a seperated preset, they would not be overwritten with the AquaTouch Update.

    It, however, is not possible to save modifications that involve deleting or editing AquaTouch's code, as this is directly editing AquaTouch itself, and will be overridden when updated.
    If your customisation requires deleting or editing AquaTouch, you'll need to re-do or add them back when a new update is installed.

    As AquaTouch contnues to change through updates, modifications that are added may get misplaced if AquaTouch undergoes a design refresh. Your additions will stay put, but they may need to be manually modified to work with the new design.





    Update Persisting Customisations

    Intro Prep Step 1 Step 2 Step 3

    Preperation

    Avoid adding anything into the AQT preset. If you do, it will be overwritten when a new AQT update is installed.

    If you've already added your own customisations

    In theory, you'd set this up before starting any edit, but if you have already made changes to AQT, you can move them out to your own preset.

    I recommend backing up your current customised AQT (very important since BTT likes to break up when deleting things), then delete everything that isn't yours from your current AQT, leaving your customisations behind in the right place. (pluck all the original AQT out of the preset and leave your changes behind).

    Then you can install a fresh copy of AQT as a seperate preset, set your personal preset as Master and set AquaTouch as an active preset.


    You might find that some of your additions have misaligned, so clean up any mishaps from your personal preset.
    from here you can follow the steps and tips in the above section!

    Adding your own Application to AquaTouch

    Not as normal.

    You can add an app normally by pressing the plus button in the bottom left corner of BTT, but you might find that whatever you add will mix with the Gesture Bar screen.
    This is because your recently added app is still tagged as a "Not Supported" app by AquaTouch.

    How to Add Applications to AquaTouch

    You can add a new app as normal (pressing the plus button in the bottom left corner of BTT, leftmost column), but make sure to exclude it from the unsupported apps list to avoid it mixing with the items from the Gesture Bar View.


    1 - Head to "Unsupported Apps" in the leftmost column
    2 - Click "Edit Conditions For Group in the rightmost column
    3 - Add a new entry to the list with "App Name is [Your App Name]"
    Your new app should no longer mix with the Gesture Bar

    TouchBar Configuration

    Every Application customised with BetterTouchTool has a setting called "TouchBar Configuration".
    This controls the behaviour of the stock Control Strip and the interaction between Stock and BTT TouchBars.

    AquaTouch's Use of TouchBar Configuration

    AquaTouch uses TouchBar Configuration to handle automatic switching between the BetterTouchTool TouchBar or the Stock TouchBar.

    It is set to either one of two options:



    • Show App Default TouchBar •• Hide Control Strip in case BTT bar is active
      This setting shows the Stock touchbar as default. (Automatically closes AquaTouch)
      Use this option if you wish to focus on the stock touchbar, but add some extra shortcuts inside AquaTouch.
    • Show BetterTouchTool Touch Bar •• HIDE macOS Control Strip
      This setting prioritises the AquaTouch UI. It opens the AquaTouch touchbar automatically, hence replaces the stock touchbar.
      Use this option when your App's stock touchbar is useless =P

    Positioning Elements within AquaTouch

    The "Display Order" Feature

    For every touch bar item, BetterTouchTool allows you to include a "Display Order".
    This is a number that gives a further heirarchy to items, ('stronger' than the order that the triggers appear in BTT)

    AquaTouch's use of Display Order


    AquaTouch uses Display Order to group and position right-aligned widgets.
    If you add a standard right-aligned widget in BTT, you might find that it won't go any further beyond the quickstrip or notification badges, no matter it's position in the trigger list.

    If you wish to add an item within the QuickStrip group, use this guide below:

    • Display order 0 contains standard app controls.
    • Display order 1 only contains a blank spacer, so that notification badges are correctly spaced out. Normally you wouldn't need to add anything here.
    • Display order 2 positions buttons to the left of the quickstrip, in with notification badges
    • Display order 3 positions buttons with the quickstrip
    • Display order 4 positions buttons to the right of the quickstrip, in with Pinned App Controls.

    The above is applied to the right-aligned "AquaTouch Main UI" only.

    Updating AQT

    Check for Modifications

    If you've modified AquaTouch within the BTT editor, you might loose changes on update if not implemented properly.
    Be aware that any modifications you have made may be lost on update!

    If you haven't edited AquaTouch in the BTT editor, you're all good to go.

    Checking for Updates

    You can check for updates at the AquaTouch Updates Thread, or see any new versions by opening the BTT Configuration window and looking in the touchbar (updates will be displayed in the white strip).

    Recommended Updating Procedure

    Following this reduces the chance of data loss, old version clutter and corruption on install.

    1 - Visit the AquaTouch Updates thread.
    2 - Read the changelog for the new version.
    3 - Download the new update. (Don't open the download yet!)
    4 - Delete the current version of AQT.
    5 - Import the new version, considering any installation notes from the changelog online.
